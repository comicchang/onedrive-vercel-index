# public/images

## Purpose
错误页面与空状态页面所用的插图资源，在 404/500 等错误页和空目录提示中引用。

## Key Files

| 文件 | 说明 |
|------|------|
| `🤐.png` | 密码保护目录提示图 |
| `😟.png` | 通用错误提示图 |
| `🔑.png` | 认证/授权错误图 |
| `🚀.png` | 加载/初始化状态图 |
| `🥴.png` | 意外错误图 |
| `🦒.png` | 空状态/无内容图 |
| `💫.png` | 其他状态图 |

## For AI Agents

### Working In This Directory
- 文件名为 emoji，在代码中通过字符串路径引用，如 `/images/🤐.png`
- 替换图片时保持文件名（含 emoji）不变
