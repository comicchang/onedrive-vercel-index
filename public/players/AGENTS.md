# public/players

## Purpose
媒体播放器品牌图标，在视频/音频预览组件中展示对应播放器的标识。

## Key Files

| 文件 | 说明 |
|------|------|
| `plyr.png` | Plyr 播放器图标（用于视频/音频） |
| `flv.png` | FLV/mpegts.js 播放器图标 |
| `iina.png` | IINA 播放器图标（macOS 外链） |
| `potplayer.png` | PotPlayer 播放器图标（Windows 外链） |

## For AI Agents

### Working In This Directory
- 这些图标由 `src/components/previews/` 中的媒体预览组件引用
- 不需要修改，除非更换播放器库
