# public/icons

## Purpose
应用导航栏图标，提供 4 种尺寸的 PNG 文件，供 `config/site.config.js` 的 `icon` 字段引用。

## Key Files

| 文件 | 说明 |
|------|------|
| `64.png` | 64×64px 应用图标 |
| `128.png` | 128×128px 应用图标（默认引用） |
| `256.png` | 256×256px 应用图标 |
| `512.png` | 512×512px 应用图标 |

## For AI Agents

### Working In This Directory
- 替换图标时保持文件名不变，或同步修改 `config/site.config.js` 中的 `icon` 路径
- 图标格式必须为 PNG，支持透明背景
